# MyHome - 부동산 중개 플랫폼

매도자와 공인중개사를 직접 연결하는 부동산 중개 플랫폼입니다.

## 주요 기능

- **매물 등록**: 매도자가 직접 매물 정보를 등록
- **중개사 매칭**: 지역 기반 공인중개사 자동 매칭
- **실시간 알림**: 매물 상태 변경 및 방문 요청 알림
- **방문 스케줄링**: 중개사-매도자 간 방문 일정 조율
- **채팅**: 실시간 1:1 채팅 기능

## 기술 스택

- **Frontend**: Flutter (iOS, Android, Web)
- **Backend**: Firebase (Firestore, Auth, Storage)
- **Authentication**: Google, Kakao 소셜 로그인

## 시작하기

```bash
# 의존성 설치
flutter pub get

# 개발 서버 실행
flutter run
```

## 빌드

```bash
# Android APK
flutter build apk --release

# iOS
flutter build ios --release

# Web
flutter build web --release
```
