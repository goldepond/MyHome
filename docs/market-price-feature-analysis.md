- 3단계 캐싱: 메모리 → 로컬저장소 → API
- 단계적 로딩: 최근 3개월 먼저 표시 후 전체 로드
- 고급 필터: 면적/층수/건축년도/가격대/검색범위

### 접근 경로 (3가지)
1. **로그인 유저**: MainPage 3번째 탭 "시세"
2. **등록 중**: Step 1 가격 입력에서 "이 아파트 최근 실거래가 보기"
3. **비로그인**: 랜딩 페이지 링크 또는 `/market-price` 직접 URL (SEO)

| 서비스 | 핵심 강점 | 타깃 |
|--------|----------|------|
| **호갱노노** | 지도 + 단지별 시세 버블, 가격 변동 그래프, 3D 일조량/조망, 200만 회원 | 투자자 |
| **네이버 부동산** | 매물 수 최다, 지도 기반 검색, 접근성 (네이버페이 통합) | 일반인 |

| **아실** | 입주민 리뷰, 실거주자 정보, 재건축 정보 | 실거주 목적 |
| **리치고** | 부동산 빅데이터, 투자 분석 | 투자자 |

- 학군, 학원가, 범죄율 시각화
- 3D 단지투어 (조망, 일조량)
- 중개라이브
- 분양/재건축/경매 정보

## 3. 사용자 불만 및 문제점

### 공통적인 UX 문제

| 문제 | 상세 |
|------|------|
| **정보 과부하** | "필터가 왜 이렇게 많아! 정신없다" |
| **비교 번거로움** | 지역 비교 시 시>구>동 반복 선택 필요 |
| **제도 정보 부재** | 근저당, 대출 가능 여부, 보증보험 확인 어려움 |
| **교육 콘텐츠 없음** | 부동산 초보자가 용어/개념 이해하기 어려움 |
| **면적 혼동** | 전용면적 vs 공급면적 vs 평 변환 헷갈림 |
| **앱 분산** | 투자는 호갱노노+아실, 자취는 직방+다방 - 하나로 안 됨 |
| **UI 품질 편차** | 앱마다 시각적 정보 표현 수준 크게 차이 |

### 핵심 인사이트
> "호가만 보고 판단하면 실수하기 쉬움. 호갱노노랑 아실을 함께 돌려봐야 실거래가랑 차이가 딱 보인다"

## 4. MyHome 차별화 전략

### 핵심 포지션
**"시세 조회"가 아니라 "판매 의사결정 도우미"**

- 호갱노노 사용자: "이 아파트 얼마지?" → 정보 소비 후 이탈
- MyHome 사용자: "이 가격에 팔 수 있을까?" → 등록으로 전환

### 차별화 방향

| 기존 서비스 | MyHome 차별화 |
|------------|---------------|
| 시세만 보여줌 | "이 가격이면 빨리 팔려요" 판매 확률 표시 |
| 단순 평균가 | 평형별/층별/향별 세분화된 적정가 제안 |
| 데이터만 나열 | "지금 등록하면 3일 내 중개사 5명 연락" 실시간 수요 표시 |
| 복잡한 필터 | 주소 하나만 입력하면 끝 (30초 시세 확인) |

## 5. 구체적 기능 아이디어

### A. 즉시 적용 가능 (현재 데이터 활용)

1. **적정 호가 계산기**
   - 평균가 대비 ±10% 범위 표시
   - "이 가격이면 평균보다 저렴해요"

2. **거래 속도 표시**
   - "이 단지 최근 거래: 매물 등록 후 평균 23일"

3. **가격 트렌드 화살표**
   - 최근 3개월 상승/하락 시각화

4. **"내 집과 비슷한 최근 거래"**
   - 같은 평수+층 거래만 필터

### B. 중기 개발 (추가 작업 필요)

5. **중개사 관심도 표시**
   - "현재 이 지역 매물 찾는 중개사 12명"

6. **예상 수수료 계산**
   - 거래가 기준 중개 수수료 자동 계산

7. **타임라인 뷰**
   - 가격 변화를 시간순으로 스와이프

### C. 장기 (데이터 축적 후)

8. **AI 적정가 추천**
   - 면적, 층, 향, 리모델링 여부 등 반영

9. **매도인 후기**
   - "저는 이 가격에 2주 만에 팔았어요"

## 6. UI 개선안 (즉시 적용 가능)

### 현재 요약 섹션
```
평균 가격
5억 2천만원
━━━━━━━━━━━
최저    최고    거래
4.8억   5.6억   15건
```

### 개선안: 판매 가이드 추가
```
평균 가격: 5억 2천만원
━━━━━━━━━━━━━━━━━━━━━

💡 이 가격대 등록 시 예상 반응

   5억 이하     →  빠른 거래 예상 ⚡
   5억 ~ 5.5억  →  평균 속도
   5.5억 이상   →  협상 여지 필요

━━━━━━━━━━━━━━━━━━━━━
      [이 가격에 매물 등록하기]
```
